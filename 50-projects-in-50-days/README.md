
# 50 Projetos em 50 Dias

Este repositório reúne uma série de **50 projetos** desenvolvidos com **HTML, CSS e JavaScript puro**, baseados no desafio _50 Projects in 50 Days_. A proposta é criar um projeto por dia para fortalecer conhecimentos práticos em **desenvolvimento front-end**, com foco em:

- Manipulação do DOM
- Animações com CSS
- Layouts responsivos
- Interatividade com JavaScript Vanilla


> ✅ Projeto criado originalmente por [Brad Traversy](https://github.com/bradtraversy) no desafio [50projects50days](https://github.com/bradtraversy/50projects50days). Todos os créditos a ele.

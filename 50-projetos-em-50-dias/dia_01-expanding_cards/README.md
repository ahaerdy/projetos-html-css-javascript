Este projeto foca na criação de **cartões expansíveis dinâmicos** utilizando uma combinação de tecnologias web fundamentais. O objetivo principal é estilizar painéis com **imagens de fundo** e títulos, organizando o layout através de **flexbox** para garantir o alinhamento adequado. Ao interagir com os elementos, o **JavaScript** gerencia eventos de clique que alternam classes específicas para disparar **transições suaves de CSS**. Esse efeito visual permite que os painéis aumentem de tamanho de forma elegante, revelando o conteúdo textual apenas após a expansão completa. Embora o exemplo utilize fotos, a estrutura é versátil o suficiente para ser adaptada em seções de **depoimentos ou perguntas frequentes**. No geral, o tutorial demonstra como unir **estilização visual e interatividade** para melhorar a experiência do usuário.

Neste projeto inicial, o objetivo é construir **Expanding Cards** (cartões expansíveis). A interface consiste em painéis que contêm uma imagem de fundo e um título correspondente. A estrutura utiliza **Flexbox** para o alinhamento de todos os elementos na tela, garantindo que os cartões fiquem posicionados corretamente lado a lado. Através de JavaScript, será implementada uma lógica para que, ao clicar em um painel, a classe do elemento seja alterada para `active`, permitindo que ele se expanda.

<p align="center">

Para garantir uma experiência de usuário fluida, são utilizadas **transições CSS** (`CSS transitions`), que proporcionam um efeito de movimento suave durante a expansão do cartão. Um detalhe importante da implementação é que o título (heading) de cada painel permanece oculto ou não aparece totalmente até que o cartão esteja completamente expandido. Além das imagens e títulos demonstrados, esta estrutura é versátil e pode ser adaptada para outros componentes, como áreas de depoimentos ou seções de perguntas frequentes (FAQs).
